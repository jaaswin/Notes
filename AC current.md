What is AC Current?

AC (Alternating Current) is a type of electric current in which the direction of flow of electric charge changes periodically.
Unlike DC (Direct Current), where electrons move only in one direction, in AC the electrons swing to and fro like a pendulum.

How AC Current is Generated

AC current is mainly produced by alternators (AC generators).

Working idea:

A coil rotates inside a magnetic field

This rotation causes a changing magnetic flux

Due to electromagnetic induction, an electric current is produced

As the coil rotates, the direction of current changes periodically

Important Terms in AC Current

1. Frequency (f)

Number of cycles completed per second

Measured in Hertz (Hz)

In India: 50 Hz

In USA: 60 Hz


This means in India, AC changes direction 50 times per second!

2. Time Period (T)

3. Peak Value (Maximum Value)

4. RMS Value (Root Mean Square Value)

This is the effective value of AC current.

Because AC keeps changing, we cannot use simple average value.
